# 万象卡

这是基于《梅花易数捷要》第 8 章“万物类象汇总（重点）”制作的万物类象记忆网页原型。

## 使用方式

直接打开 `index.html` 即可使用。

## 已包含

- 首页入口：八卦分类学习、类象卡片记忆、学习计划
- 账号同步：预留 Supabase 邮箱登录与云端同步配置，适合部署到手机端使用
- 学习目标：设置完成天数，自动计算每日新学量
- 复习计划：按 1、2、4、7、15、30 天间隔生成 7 日计划
- 八卦分类学习：点击乾、坤、震、巽、坎、离、艮、兑进入子页面
- 单卦子页面：按现代易象与传统类象分类展示所有相关类象
- 类象卡片记忆：事物类象反推卦象，可标记陌生、模糊、熟悉
- 不再复习：用户选择后，该卡片不再纳入复习计划
- 专项牌组：从单卦子页面进入时，只练当前卦
- 模糊查词：支持“大地 -> 坤/田野/平地”等近义匹配
- 相关卦提示：搜索“土”时提示坤、艮，但不再直接展开坤艮全部条目
- 搜索添加：未收录词条可直接选择卦象并加入“我的类象”
- 我的类象：用户可在每个卦页面添加自定义类象，添加前会检查当前卦是否已有相同词条
- 查词模式：搜索事物、身体、疾病、方位、颜色、五行、卦名、自定义类象等
- 云端数据：配置 `sync-config.js` 后，同步学习进度、复习日期、学习目标和自定义类象
- 本机缓存：未登录或离线时暂存数据，登录后可同步到账号
- 黑白水墨 UI：铺满式灰白背景、圆弧墨线卡片、黑白灰色卡、国画梅花与柔和灰阶花瓣装点

## 文件说明

- `index.html`：网页入口
- `sync-config.js`：账号同步配置
- `supabase-schema.sql`：Supabase 数据表与权限策略
- `styles.css`：界面样式
- `app.js`：结构化数据与交互逻辑
- `assets/ink-landscape-bg.png`：水墨山水背景图
- `assets/plum-wash.png`：国画梅花装饰图
- 上一级目录 `wanwu-leixiang-ocr.txt`：第 8 章 OCR 提取稿

## 校对说明

原 PDF 为扫描版，OCR 可能存在误字。网页里的学习数据已做了初步整理，适合先试用学习流程；若要作为正式资料，建议逐页对照原 PDF 再校订术语。

如果页面已经在浏览器中打开，刷新 `index.html` 即可看到新的水墨风格。Codex 内置浏览器对直接刷新 `file://` 页面有限制，因此此处主要做了文件级校验。
